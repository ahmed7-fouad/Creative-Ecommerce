# Creative Ecommerce

A fully responsive modern E-commerce web application built using Vanilla JavaScript, HTML, and CSS.

This project was built from scratch without using frontend frameworks, focusing on real-world frontend concepts like state management, authentication flow, local storage persistence, API handling, and dynamic UI interactions.

---

## 🚀 Features

- 🔐 Authentication System
- 🛒 Shopping Cart Functionality
- ❤️ Wishlist Page
- 📦 Product Details Page
- 💳 Checkout System
- 📡 API Requests & Dynamic Data Fetching
- 📁 Local Storage State Persistence
- ✅ Form Validation
- 🔄 State Recovery after Refresh
- 📱 Fully Responsive Design
- ⚡ Smooth User Experience

---

## 🛠️ Technologies Used

- HTML5

- JavaScript (Vanilla JS)
- REST APIs
